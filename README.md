This was created as part of the final project for CS 528 at UMass.

**Beat Saber **
This clone allows users to play the classic Beat Saber game using left, right, up, down, forward, backward gestures with an in-hand device.

We collected IMU data, trained an SVM model, and translated motion into gestures in-game. 


Hardware Setup:
This system used an ESP32, hookup to an IMU, which was connected to a laptop for power.
